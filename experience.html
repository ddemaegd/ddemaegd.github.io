Active Development: (Oct. 2014 - Present)

  A 2D cyberpunk side-scroller shooter. Since October 2014, I have been developing this game on my own in my spare time. This would include all of the graphics, programming, and sound effects. I wanted to make this game simple but unique. A cyberpunk genre set in the Middle Ages. Technology has improved immensely but the world politics have never changed.
